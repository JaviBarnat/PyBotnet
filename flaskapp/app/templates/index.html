Sobre PyBotnet

Botnet desarrollada con fines educativos basada en protocolo IP contra equipos de la familia Windows. Tanto los bots como la aplicación web establecen una comunicación segura con el servidor de procedimiento remoto para la realización de las distintas tareas. El trabajo ha sido desarrollado utilizando Python como lenguaje de programación, gRPC como sistema de llamada a procedimiento remoto (RPC) para el servidor 'Command & Control' que utiliza como transporte HTTP/2 y Protocol Buffers como lenguaje de descripción de interfaz.

La aplicación web está escrita en Python y Flask.
Los bots están escritos en Python y compilados con el módulo PyInstaller.

¿Qué hace realmente PyBotnet?

Es la aplicación web se encarga de gestionar todos los aspectos de administración del servidor 'Command and Control' o C&C. Entre estos aspectos destaca la posibilidad de ver qué bots se han conectado con el servidor, descargar los logs generados a lo largo del tiempo en el servidor y mostrar su ubicación en un mapa global.
Su principal funcionalidad es la gestión de tareas de los bots de la botnet.

📋 Tareas

La principal funcionalidad de una botnet es la utilización del bot y de las máquinas infectadas con un objetivo determinado. A continuación se recogen las tareas presentes que puede realizar el bot:

Tarea CMD

La tarea CMD permite ejecutar el comando deseado en la máquina remota y este devolverá el resultado de la ejecución.

Tarea UPLOAD

La tarea UPLOAD permite subir un fichero desde la máquina del servidor a la máquina donde reside el bot.

Tarea DOWNLOAD

La tarea DOWNLOAD permite bajar un fichero desde la máquina remota a la máquina servidor.

Tarea SCREENSHOT

La tarea SCREENSHOT realiza una captura de pantalla de la máquina remota y envía el resultado al servidor.

Tarea SHUTDOWN

La tarea SHUTDOWN permite apagar el bot de manera remota.

Tarea DELETE

La tarea DELETE permite eliminar el bot de manera remota.

📝 Logs

El apartado Logs muestra los logs recogidos, tanto sobre los bots como sobre el servidor. Desde esta página puede acceder a ellos y descargarlos directamente.

📂 Archivos

El apartado muestra todos los elementos que hayan sido descargados de las máquinas remotas a través de la tarea DOWNLOAD. Presenta la estructura de carpetas y ficheros creada en el servidor así como su posibilidad de descargarse los ficheros directamente desde el servidor al cliente.

🌍 Mapa

El apartado Mapa muestra un mapa global. En este mapa, y siempre que haya algún bot registrado, podremos geolocalizarlo y estará marcado con un indicador. Si se pulsa sobre el indicador podremos acceder a una mayor información acerca del dispositivo.
